Welcome Git

Git 是一個分散式版本控制系統，最初由 Linus Torvalds 創建，於 2005 年以 GPL 授權條款發布。最初的目的是以一種更好的方式管理 Linux 內核開發。

Git 是一種用於跟踪文件更改的工具。它可讓您記錄文件的歷史，以便您可以輕鬆地回滾更改或查看更改的差異。Git 還可讓您與其他開發人員協作，並將您的工作與遠端存儲庫同步。

Git 具有許多優點，包括：

  • 它是分散式的，這意味著您可以離線工作，並在有網路時將您的更改與遠端存儲庫同步。
  • 它是可靠的，它使用哈希算法來跟踪文件的更改。
  • 它是靈活的，它可用於跟踪任何類型的文件。
